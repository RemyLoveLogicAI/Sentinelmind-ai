Core Commands

POST /api/command/:command

Execute core system commands

Available Commands:
- analyze-moment: Analyze current moment for patterns
- ground-me: Trigger grounding protocol
- scramble-this: Detect and neutralize NLP patterns
- update-case-file: Log interaction to memory
- extract-what-matters: Summarize key evidence

Authentication & Headers

All API requests support the following headers:

Content-Type: application/json
X-User-Id: your_user_id (optional, auto-generated if not provided)
X-Session-Id: session_id (optional, auto-generated if not provided)

Rate Limiting

  • 30 requests per minute per user for API endpoints
  • 100 requests per minute per user for general endpoints
  • Rate limit information included in response headers

Error Codes

400 - Bad Request (Invalid input)
404 - Not Found (Endpoint doesn't exist)
429 - Too Many Requests (Rate limited)
500 - Internal Server Error
503 - Service Unavailable (Database/KV down)

Error Response Format:

{
  "success": false,
  "error": "Error message description",
  "timestamp": "2025-01-16T12:00:00.000Z"
}
