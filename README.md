# SentinelMind AI

**SentinelMind AI** is a cutting-edge, open-source hypnosis warfare and defense app powered by AI.  
It uses GPT-4, real-time contextual awareness, and modular multi-agent logic to simulate hypnosis techniques and protective countermeasures.

## 🔥 Features
- 🌀 Hypnosis Offense + Defense Modes
- 🤖 GPT-4 AI Script Generation
- 🧠 Multi-Agent Role Simulation (Chief, Captain, Forensics)
- 🔒 Subliminal Detection + Grounding Systems
- 📜 Session Logs & Script Editor

## 🚀 Tech Stack
- Frontend: React + Tailwind
- Backend: Node.js + Express
- AI: OpenAI API (GPT-4)
